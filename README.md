### Prerequisites
- macOS (Mac mini recommended)
- Rust 1.70+

### Installation

#### Step 1: Install Rust Daemon

```bash


git clone https://github.com/sacaslabs/sacas-daemon.git

curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

source $HOME/.cargo/env

brew install git rust python3

cd sacas-daemon

# Build the daemon
cargo build --release

# Run the daemon
./target/release/sacas-daemon
```

The daemon will:
- Create a config file at `~/.sacas/config.toml`
- Start the gRPC server on `localhost:50051`
- Begin mining Entropy
- Probe network topology every 60 seconds

#### Step 2: Install MCP Server

```bash
cd sacas-mcp-server

# Install dependencies
npm install

# Build TypeScript
npm run build

# Test the server
npm start
```

#### Step 3: Configure Claude

Add to your Claude Desktop config (`~/Library/Application Support/Claude/claude_desktop_config.json`):

```json
{
  "mcpServers": {
    "sacas": {
      "command": "node",
      "args": ["/path/to/sacas-mcp-server/build/index.js"]
    }
  }
}
```

Or for global installation:

```bash
npm install -g .
```

Then configure:

```json
{
  "mcpServers": {
    "sacas": {
      "command": "sacas-mcp-server"
    }
  }
}
```
